Acadalyze - eLMS WebApp
Project Overview

Acadalyze is a comprehensive eLearning Management System (LMS) for online education. It supports course creation, student management, and interactive learning.

Key Features
  • User authentication and role management
  • Real-time notifications using WebSocket
  • Real-time messaging using WebSocket
  • Interactive assignments and quizzes
  • Progress tracking and analytics
  • Course creation and management
Technology Stack
Spring Boot Thymeleaf WebSocket PostgreSQL HTML5 jQuery Bootstrap 5 Java
