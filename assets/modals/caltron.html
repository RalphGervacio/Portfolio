Project Overview

CALTRON is a lightweight academic tool designed for students and educators. Unlike typical advising systems that are online-heavy and opaque, CALTRON provides offline-capable grade computation with a transparent weighted average formula and adaptive learning suggestions powered by machine learning.

Key Features
  • Grade computation using Written Works, Performance Tasks, and Exams
  • Customizable weight percentages
  • Dynamic grade and subject selection (AJAX powered)
  • Real-time results with reset option
  • Interactive UI with math animations
  • Offline and Online-ready, lightweight design
Project Highlights
Instant Calculations

Students can compute grades in real time with a single click, ensuring immediate feedback without waiting for backend processing.

User-Friendly Design

Clean, responsive interface with Bootstrap 5 and animated math symbols for an engaging learning experience.

Smart Learning Nudges

Suggests areas of improvement, helping students focus on Written Works, Performance Tasks, or Exams as needed.

Reliable & Lightweight

Works offline and online, requires minimal resources.
